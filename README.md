# Restaurant Order API

API REST para gerenciamento de pedidos de um restaurante.

O sistema permite que usuários se autentiquem, visualizem o cardápio,
criem pedidos e acompanhem o status. Administradores podem gerenciar
pedidos e atualizar seus estados.

## 🚀 Tecnologias
- Java 17
- Spring Boot
- Spring Security + JWT
- JPA / Hibernate
- PostgreSQL
- Maven
- Docker (futuro)
- GitHub Actions (CI)

## 📦 Funcionalidades
- Cadastro e autenticação de usuários
- Listagem de itens do cardápio
- Criação de pedidos
- Atualização de status do pedido
- Histórico de pedidos do usuário

## 🧱 Arquitetura
Arquitetura em camadas:

- Controller
- Service
- Repository
- Domain
- DTO
- Exception
- Security
